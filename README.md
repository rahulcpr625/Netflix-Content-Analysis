# Netflix Content Analysis & Recommendation System
# Overview
This project focuses on analyzing Netflix Movies and TV Shows dataset to uncover meaningful insights and trends. The system performs data cleaning, exploratory data analysis (EDA), visualization, and machine learning-based modeling to understand content distribution and improve recommendations.

# The project demonstrates strong skills in data analytics, feature engineering, and predictive modeling.

# Key Features

* Processed and cleaned 8,000+ Netflix records using Python

* Performed Exploratory Data Analysis (EDA) on 15+ features

* Analyzed genre distribution, ratings, release trends, and country-wise content

* Built a content-based recommendation system using TF-IDF and cosine similarity

* Developed a Random Forest classification model to predict Movie vs TV Show

* Created visualizations to generate actionable insights

# Tech Stack

* Scikit-learn

* TF-IDF Vectorizer

* Random Forest Algorithm
